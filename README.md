<p align="center">
  <h3>Sobre mim ✨️</h3>
</p>

- Desenvolvedor de sistemas e automações em JavaScript com mais de 2 anos de experiência, focado no ecossistema de backend e no desenvolvimento de soluções de alta performance. 

- Minha trajetória começou com o desenvolvimento de bots para WhatsApp, um ramo onde sigo ativo até hoje. Meu maior marco na comunidade foi o desenvolvimento do HutaoBot-MD, alcançando e impactando grande parte dos usuários desse ecossistema.

### Meu foco principal:
* **Código Artesanal:** Crio scripts com cerca de 90% do código feito inteiramente à mão, aplicando dedicação em cada linha lógica.
* **Otimização Extrema:** Gosto de focar em testes e experimentos complexos para garantir o máximo de desempenho e eficiência nos sistemas.
* **Inteligência sem Excessos:** Priorizo códigos inteligentes estruturados pela lógica humana em vez do uso massivo de IA. É um gosto pessoal manter o desenvolvimento técnico sob controle analítico real.

### Projetos Desenvolvidos:
* Criação de APIs robustas e microsserviços velozes.
* Desenvolvimento de automações completas e scripts customizados.
* Gerenciamento e arquitetura de bots de alta escala.
